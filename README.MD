# Personal Finance Management Application

Project ID: UY6758GH

## Description
This is a Python-based command-line application that helps users manage
their personal finances by tracking income, expenses, generating reports,
setting budgets, and backing up data.

## Technologies Used
- Python
- SQLite3

## Features
- User registration and login
- Income and expense tracking
- Monthly and yearly reports
- Category-wise budgeting with alerts
- Database backup and restore

## Installation
1. Install Python 3.x
2. Download project files
3. Open terminal in project folder
